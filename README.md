# README

## Running Locally

Make sure you have Ruby installed.

```sh
$ git clone git@github.com:Azfletch/blog.git
$ cd blog
$ bundle install
```

Now :-

```sh
$ rails s

```

The blog should now be running on [localhost:4000](http://localhost:4000/).

## Running Tests

To run the test suite use the following command

```sh
$ rails spec

```

## Database setup

```sh
$ rails db:create
$ rails db:migrate

```
